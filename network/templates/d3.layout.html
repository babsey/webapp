Network layout

  1. Add new nodes or new links and set their status.
  2. Explore the network layout.
  3. Explore the connectivion matrix.
  4. Modify nodes and links.
  5. Run the simulation.
  6. View the results.
Each successful simulation appears in the history list. The same list can be used to retrieve past simulations together with the results. The recorded events (spikes, membrane potential etc.) are plotted online. They can also be downloaded as text files.

Optional:

Add comments to each simulation describing what you did it, why you did it, what were the results etc. This will help also when retrieving old simulations.

Give some feedback in the forum, if you want to share it.

  • HINT: All nodes can be dragged around.
  • Create a new node: Doubleclick any space in layout.
  • Create a new link: Hold clicking a node and move the mouse to next node.
  • Delete node or link: Select a node/link and press the key 'Delete'.
  • Additional options: Alternate the options for your sake.
  • Blue links with black arrow - excitatory connection
  • Red links with black dot - inhibitory connection
