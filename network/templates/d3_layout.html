  • Create a new node: Doubleclick any space in layout.
  • Create a new link: Hold clicking a node and move the mouse to the next node.
  • Delete node: Focus a node with the mouse and press the key 'Delete'.
  • Delete link: If it'able to change links, doubleclick a cell in connection matrix to create or delete a link.
  • HINT: All nodes can be dragged around. (with Ctrl-Pressing)
  • Blue links with black arrow - excitatory connection
  • Red links with black dot - inhibitory connection
  • Additional options: Modify the options for your sake and click button 'Options' again to save the changes.
